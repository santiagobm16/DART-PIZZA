2. ¿Quién lo usará?
   En Nuestro caso los roles seria el de Administrador y Cliente.
3. ¿Qué pasaría si no existiera?
   Esto limita sus ventas ya que estos en un entorno donde las aplicaciones digitales dominan el mercado.
